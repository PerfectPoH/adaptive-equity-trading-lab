## Validated source pattern

The practical public route is hybrid:

1. Discover candidate events from public calendars/search.
2. Require pre-event proof from issuer press releases, SEC filings, or issuer-hosted documents.
3. Keep non-success events such as CRLs and delays.
4. Run only a micro-backtest after a separate gate.
