# LowVol Tradability Trial 001 - Preregistration

`TRIAL-LOWVOL-TRADABILITY-001` tests a long-only, non-earnings hypothesis using only existing Databento daily OHLCV artifacts.

Hypothesis: among the available small-cap universe, selecting the lowest realized-volatility symbol that passes basic tradability filters should improve robustness relative to event-driven ideas that depended on catalyst direction.

This trial is intentionally price/volume-only. It does not claim fundamental quality because PIT fundamentals are not available.
