# Candidate 018 FMP Micro-Probe Gate

This gate authorizes exactly one bounded Financial Modeling Prep component probe after the user added `FMP_API_KEY` locally.

The run may verify active OHLCV, SPY/IWM benchmark OHLCV, delisted-list access, and terminal delisted OHLCV access. It may not retain raw payloads, build a dataset, run a backtest, run portfolio selection, run Kronos inference, promote a strategy, paper trade, or live trade.
