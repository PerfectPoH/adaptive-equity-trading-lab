# Norgate Data Gate 001

This gate authorizes one bounded local provider probe after the Norgate Data trial was installed and databases were downloaded by the user.

The probe may inspect the Python package, enumerate a small set of available database names/watchlists, and request a tiny sample of survivor-bias-aware daily price data for active and delisted US equities if the local Norgate installation exposes it.

This gate does not authorize a strategy backtest, parameter sweep, paper trading, live trading, promotion, or financial performance claim.

The trial is expected to be limited to roughly two years of history, so any downstream artifact must remain trial-limited until a longer licensed dataset is available.
