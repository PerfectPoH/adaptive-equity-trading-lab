# Candidate 016 Tiingo Micro-Probe Gate

This gate authorizes one bounded Tiingo component probe for the Hybrid Free Data Mesh path.

It may make up to eight Tiingo HTTP requests if `TIINGO_API_KEY` exists in `.env`. It may write only derived summaries and a final decision. It does not authorize raw payload retention, dataset construction, Candidate 012 backtesting, Kronos inference, portfolio selection, paper trading, live trading, or promotion.
