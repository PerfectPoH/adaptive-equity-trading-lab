# Public Catalyst Hybrid Event Panel Gate 001

This gate defines the only admissible free/public route for a Catalyst sleeve mini-panel.

The prior audit found that no observed public source is promotable by itself. The only defensible route is hybrid:

1. Use public/freemium calendars only to discover candidate biotech/PDUFA events.
2. Require event-level pre-event proof from issuer press releases, SEC filings, or archived pages.
3. Explicitly include negative, delayed, withdrawn, missed, and unresolved events.
4. Keep the panel non-promotable until completeness and retention are proven.

No market-data download, backtest, paper trading, live trading, or promotion is authorized by this gate.

## Scope

The first panel may contain 10 to 20 events. It must be small enough for manual audit, but diverse enough to include approvals, CRLs/rejections, delays/extensions, and unresolved/no-action outcomes if available.

## Output requirement

Every event row must carry a source trail:

- discovery source
- pre-event proof URL
- proof publication date
- event date
- event type
- ticker/company identifier
- drug/asset
- indication
- outcome if known
- outcome source if known
- admissibility label
