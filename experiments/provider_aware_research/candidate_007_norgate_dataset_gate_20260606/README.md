# Candidate 007 Norgate Dataset Gate 001

This gate authorizes one bounded local Norgate dataset build.

It may read local Norgate symbols and daily price timeseries for active and delisted US equities, then write a manifest, security master, prices file, tradability report, validation report, and final decision.

It does not authorize internet market-data downloads, strategy backtests, Kronos inference, portfolio selection, promotion, paper trading, live trading, or alpha/performance claims.
