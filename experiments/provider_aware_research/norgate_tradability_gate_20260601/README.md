# Norgate Tradability Gate 001

This gate freezes a tradability filter before rerunning the Norgate Portfolio Candidate 002 diagnostic.

The previous Norgate trial backtest was archived because the apparent return was dominated by one outlier and the generated trades included sub-dollar/micro-penny contamination. This gate does not tune the strategy to improve returns. It defines minimum operating-quality constraints that must be applied before any rerun is interpreted.

The rerun remains trial-limited, non-promotable, and diagnostic-only.
