# Candidate 007 Dataset Audit Gate 001

This gate authorizes a read-only audit of the already-built Norgate dataset.

It may compute coverage, active/delisted mix, date-span, row-count, liquidity, and tradability diagnostics.

It does not authorize provider queries, market-data downloads, strategy backtests, Kronos inference, portfolio selection, promotion, paper trading, live trading, or alpha/performance claims.
