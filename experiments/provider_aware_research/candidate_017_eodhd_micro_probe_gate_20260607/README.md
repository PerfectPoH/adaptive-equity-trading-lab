# Candidate 017 EODHD Micro-Probe Gate

This gate authorizes one bounded EODHD component probe for the Hybrid Free Data Mesh path.

It may make up to six EODHD HTTP requests if an accepted EODHD key exists in `.env`. It may write only derived summaries and a final decision. It does not authorize raw payload retention, dataset construction, Candidate 012 backtesting, Kronos inference, portfolio selection, paper trading, live trading, or promotion.
