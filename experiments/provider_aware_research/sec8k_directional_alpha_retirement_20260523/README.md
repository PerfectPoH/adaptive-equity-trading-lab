# SEC8K Directional Alpha Retirement

Status: SEC8K_DIRECTIONAL_ALPHA_RETIRED

SEC 8-K Item 2.02 remains documented as a volatility and liquidity regime, but
all long-only directional alpha variants tested so far are retired:

- SEC8K Tape Oracle 001
- SEC8K Tape Oracle 002
- SEC8K Predrift 001

Future SEC8K long-only directional trials are blocked unless a separate
point-in-time external direction source is pre-registered first. Short trials are
out of scope permanently.
