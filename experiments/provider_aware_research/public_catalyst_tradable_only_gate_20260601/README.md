# PUBLIC-CATALYST-TRADABLE-ONLY-GATE-001

This gate allows construction of a strict tradable-only subset from the expanded public catalyst panel and one strict as-of micro-backtest rerun.

The gate is designed to answer a narrow question: does the catalyst sleeve still look interesting after removing private companies, unresolved date-aware symbol mappings, non-local price coverage, and immature events as of 2026-06-01?

No provider query, market-data download, parameter optimization, promotion, paper trading, or live trading is allowed.
