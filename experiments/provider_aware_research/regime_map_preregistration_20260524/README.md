# Regime Map Preregistration - 2026-05-24

`REGIME-MAP-001` is a diagnostic-only regime classifier and failure-attribution run.

It uses existing daily OHLCV artifacts and existing archived trade logs. It does not query providers, download market data, backtest a new strategy, optimize thresholds, execute paper/live trades, or promote any strategy.

The purpose is to ask whether archived long-only failures were uniformly bad or concentrated in observable market regimes.
