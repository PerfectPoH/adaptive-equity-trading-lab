# Candidate 008 Norgate OOS Baseline Gate 001

This gate authorizes one fixed diagnostic backtest on the audited Candidate 007 Norgate dataset.

It freezes a two-sleeve baseline: 60-day momentum and 5-day mean reversion, with as-of tradability filtering, 500 bps round-trip costs, and a fixed IS/OOS split.

It does not authorize provider queries, market-data downloads, Kronos inference, portfolio search, parameter sweeps, promotion, paper trading, live trading, or alpha/performance claims.
