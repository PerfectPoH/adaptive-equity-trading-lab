# Candidate 017 EODHD Micro-Probe Rerun Gate

This gate authorizes exactly one bounded EODHD component probe rerun after the user added `EODHD_API_KEY` locally.

The rerun may verify delisted list access, terminal delisted OHLCV access, and active adjusted EOD access. It may not retain raw payloads, build a dataset, run a backtest, run portfolio selection, run Kronos inference, promote a strategy, paper trade, or live trade.
