# Architecture

```text
Market Data
-> Data Snapshots
-> Feature Engineering
-> Scanner
-> Temporal Split
-> Label Builder
-> ML Model
-> Signal Engine
-> Risk Manager
-> Execution Simulator
-> Backtest
-> Metrics
-> Experiment Log
-> Streamlit Dashboard
```

Milestone 1 keeps the runtime small. News, paper trading, Graphify, vault workflows, and institutional validation are roadmap items, not MVP code.
